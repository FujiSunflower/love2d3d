Love2D3D

概要 / Overview

画像から3Dモデルを作り、スキニングもするBlenderのアドオンです。
Blender addon to add 3D object from 2D image or its armature.

使い方 / How to use

トラブルシューティング / Troubleshooting

アドオンを読み込むとエラーが出る。 / Error happen on loading the addon.

図の手順でアドオンを保存するフォルダを指定してください。
You look at the figure and set path for addon.
それでも読み込めない場合は、Blender公式サイトからBlenderをダウンロードして、アドオンを読み込んでみてください。
If error happen yet, download Blender from the official and try to load the addon once.
